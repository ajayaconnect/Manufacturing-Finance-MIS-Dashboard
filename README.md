# 📊 Manufacturing Finance MIS Dashboard

## 🔍 Project Overview
This project is an enterprise-level Finance MIS system built in Microsoft Excel for a manufacturing organization.

The objective was to simulate a real-world Finance MIS Executive role by developing structured financial reporting, variance analysis, and integrated financial statements to support management decision-making.

---

## 🎯 Business Objective
To design a dynamic MIS reporting system that enables:

- Daily / Monthly performance monitoring
- Budget vs Actual variance tracking
- Financial statement preparation
- Cash flow visibility
- Executive-level dashboard reporting

---

## 🏗 Project Architecture

The project consists of:

### 1️⃣ Data Layer
Structured datasets including:
- Sales Data
- Production Cost Data
- Expense Data
- Budget Data
- Asset Data
- Inventory Data

All data converted into structured Excel Tables for dynamic calculations.

---

### 2️⃣ Calculation Layer (MIS_Calculation)
- Total Revenue
- Production Cost
- Gross Profit
- Operating Expense
- Net Profit
- Net Profit %
- Monthly Dynamic Filtering using SUMIFS

---

### 3️⃣ Financial Statements

#### 📑 Profit & Loss Statement
- Revenue
- Cost of Goods Sold (COGS)
- Gross Profit
- Operating Expenses
- Net Profit

#### 🏦 Balance Sheet
- Current Assets (Cash, Inventory, Receivables)
- Fixed Assets
- Liabilities (Payables, Loan)
- Equity (Balancing Figure)
- Auto-validation: Assets = Liabilities + Equity

#### 💰 Cash Flow Statement
- Operating Activities (Net Profit + Depreciation)
- Investing Activities (Asset Purchase)
- Financing Activities (Loan)
- Closing Cash linked to Balance Sheet

---

### 4️⃣ Budget vs Actual Analysis
- Department-wise Actual Expense
- Department-wise Budget
- Variance Calculation
- Variance %
- Conditional Formatting (Over / Under Budget)
- Monthly Trend Comparison

---

### 5️⃣ Interactive Dashboard

The dashboard includes:

- KPI Cards (Revenue, Profit, Expense, Variance)
- Monthly Revenue Trend (Line Chart)
- Expense Distribution (Donut Chart)
- Budget vs Actual Comparison (Column / Line Chart)
- Slicers for Month & Department filtering
